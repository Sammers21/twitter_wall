## How to run it?

**1. clone it**
```
$ git clone https://github.com/Sammers21/twitter_wall
```
**2. enter**

```
$ cd twitter_wall
```
**3. run it**

```
$ ./gradlew run
```

**4. enjoy in browser**

the program run on the localhost:8080
